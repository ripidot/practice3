add->commit->pushの流れ

git add . # すべてのファイル追加
git add hoge.txt # hoge.txtを追加

git commit # vimエディタ上で編集
git commit -m "huga" # コメントをhugaにしてcommit

git push origin main # originの内容をmainにpush
